How to Make a Basic Ring in Onshape

Introduction and Purpose

These instructions guide the student through the process of creating a basic ring using Onshape, a computer-aided design (CAD) software. The purpose is to demonstrate basic 2D and 3D techniques to 1st year mechanical engineering students to help them make a parametric ring. This document assumes the user has basic familiarity with a mouse and keyboard.

Necessities

  • Computer (at least 8GB of RAM recommended)
  • Stable Internet connection
  • Active OnShape account

Important Warnings and Notes

  • Units: The default unit setting in Onshape is inches. All dimensions entered will be in inches unless the unit settings are changed (which is not covered in these instructions).
  • Troubleshooting: If an error occurs, use the Undo command (Ctrl + Z or the curved arrow icon at the top) to revert to the previous step.
  • Reference: Make sure the correct sketch plane is selected before starting a new sketch.

Part 2: Drawing the Ring

  • This section creates the cross-section and size of the ring.

Step

Instruction

Detail

2.1

Draw the Outer Diameter Circle.

Select the Center Point Circle  tool. Click once on the Origin  (the center point where the axes intersect) and drag the mouse outward to the necessary diameter. Click again to place the circle.

Tip: If you accidentally draw a line you don ’t need, you can press Ctrl-Z or click the line or shape you created and press delete.

2.2

Define the Revolve Axis Line.

Select the Line  tool. Click once on the negative X-axis  (left of the origin) to start the line. Drag the to create a vertical line of any length, and click to end the line. Note: It doesn ’t matter how big the line is, just that it ’s vertical.

2.3

Exit the Line Tool.

Click the Line  tool icon again, or press the Esc  key, to stop creating lines.

2.4

Dimension the  Ring 's Radius.

Select the Dimension  tool. Click on the center point (origin)  of the circle. Next, click on the bottom endpoint  of the vertical line created in Step 2.2.

2.5

Choose the Inner Radius.

Click on the dimension arrow and type in the numerical value for the inner radius (e.g., 0.375  for a typical ring size), and press Enter .

Part 3: Revolving the Cross-Section

Step

Instruction

Detail

3.1

Click the Revolve Feature.

Click the Revolve  tool icon located in the toolbar.

3.2

Select the cross-section.

Ensure the circle is selected under " Faces and sketch regions to revolve .”(It is usually already selected.)

3.3

Select the Revolve Axis.

Click the red text that says " Revolve axis .”Then, click the vertical line  created in Step 2.2. The circle will instantly revolve 360 degrees to form the ring.

3.4

Confirm the sketch.

Click the green checkmark  to complete the Revolve feature and create the final ring model.

3.5

Export

Right-click on the ring. Find and click export . In the export menu under Format,  find and select STL.  You can now click Export.

Part 4: Review

  • Viewing: Use the arrows around the viewing cube to rotate the model and view the ring. You can also use the right mouse button to move the model around, with Ctrl and Alt used to change the type of movement. You can use the scroll wheel to zoom in and out as well. These settings can also be adjusted according to personal preference.

  • Note: Onshape automatically saves all changes.

Checklist for Completion

  • A new  document was created and given a title
  • View selected as the Right Plane
  • New Sketch started
  • Center Point Circle drawn
  • Revolve Axis line drawn
  • Inner Radius set
  • Cross-section revolved around the correct axis
